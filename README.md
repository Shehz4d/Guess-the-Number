# Guess the Number

A responsive number guessing game built with HTML, CSS, and JavaScript.  
Guess the secret number between 1 and 20. The game tracks your score and highscore.

## Features

- Guess the secret number between 1 and 20.
- Responsive design for mobile and desktop.
- Background changes color on win or loss.
- "Again" button: start a new round without resetting highscore.
- "Restart Game" button: reset the game including highscore.

## How to Play

1. Enter a number in the input box.
2. Click **Check!** to see if you guessed correctly.
3. Score decreases on wrong guesses.
4. Click **Again!** to start a new round.
5. Click **Restart Game** to reset everything including highscore.
